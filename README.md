# Project Title

Simple library to get today date

## Description

When you dont know what day today, use this library.

### Dependencies

* Node.js 15+

### Installing

* npm i today-day

### Executing program

* How to run the program

```
const todayDay = require('today-day')

todayDay.today();

todayDay.todayLowerCase();

todayDay.todayUpperCase();
```
