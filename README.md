# Project Title

Simple library to get today date

## Description

When you dont know what day today, use this library.

## Getting Started

### Dependencies

* NodeJS 15+

### Installing

* npm i today-day

### Executing program

* How to run the program
* Step-by-step bullets
```
const todayDay = require('today-day')
todayDay.today();

```
